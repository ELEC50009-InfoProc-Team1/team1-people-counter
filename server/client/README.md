# People counter client

All `tsx` files can be found in [`./app`](/app).

The client uses the following tools:

- React as the UI framework
- Vite (Rollup) to build
- React Router for routing (in single-page-application mode)
- Material UI for general UI components.

## Devleopment usage

1. Install dependencies

    ```bash
    npm i
    ```

2. Start the development version of the server with

    ```bash
    npm run dev
    ```

## Notes

### Routing

I'd recommend having a look at the React Router documentation before having a go at anything to do with routing as it's not immediately intuitive but makes sense after a while.
